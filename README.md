# pacel-react-example

Simple React project with parcel

## Install Packages

```bash
$ npm install
$ pip install Flask
```

## Build

```bash
# Run dev server
$ npm run dev

# Build
$ npm run build

# Build for flask
$ npm run flask-build
```

## Run Server

```bash
$ FLASK_APP=server.py flask run
```
